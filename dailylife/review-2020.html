# 复盘（2020）

# 9月

# 从外到内

我不需要改变脾气再来改变行为，那样太慢。
我可以直接改变应对方式（行为），获得比原本的应对方式更多的收益后，慢慢地就会改变脾气，从外到内。

目的：减少低收益的精力消耗，把精力放在对我价值（是价值，不是利益）最大化、最重要的事情上。

# 优化

记录-归类-分析-改变
觉知-预警-取舍-优化
现在做得差，不要紧。良性循环，就可以。

你得到了一个方法，没有坚持下来，不一定说这个方法就是有问题，可能只是不够适合你。
尝试优化到适合你的版本，一点点拓展自己的能力边界。

不要把计划做得太“完美”。
要有合理的容错率。（比如时间计划中留出一段空白的时间）
正视自己普通的人性，越简单、好执行的方法越好。一次一次改进它。

# 价值

目标不等于价值。

价值不会让人觉得痛苦，目标会。
如果不清楚自己的想实现的价值是什么，在追逐目标（尤其是实现起来比较吃力的目标）的过程中就难免会感到痛苦。

如果目标围绕着价值拟定，无论成功或失败，都是一个逐渐实现价值的过程。
成功很好，失败也是积累和修炼。

我想实现的价值是什么呢？

同理，价格不是价值，不是有了价格就叫做有价值。

很多事情，不是赚钱，是在拿时间换钱。

什么样的工作，你能掌握主动权，并且积累资源？

结合价值，找到它，追求它。
